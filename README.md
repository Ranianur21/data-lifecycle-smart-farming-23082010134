# 🌾 Data Lifecycle Smart Farming

Implementasi Data Lifecycle Management menggunakan dataset sensor IoT pertanian dari Kaggle.  
Tugas 2 — Big Data & IoT 

## 📊 Dataset

- **Sumber**: [Kaggle — Smart Farming Sensor Data for Yield Prediction](https://www.kaggle.com/datasets/atharvasoundankar/smart-farming-sensor-data-for-yield-prediction)
- **Jumlah Data**: 500 baris, 22 kolom
- **Variabel Utama**: Soil Moisture, Temperature, Humidity, pH, Rainfall, NDVI Index, Yield

---

## 🚀 Cara Menjalankan Dashboard

```bash
pip install streamlit pandas matplotlib seaborn plotly
cd dashboard
streamlit run streamlit_app.py
```
